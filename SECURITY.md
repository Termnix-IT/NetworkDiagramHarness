# Security

This project generates diagrams from user-provided infrastructure definitions.

## Sensitive Data

Do not publish real network or server configuration in this repository. Treat the following as sensitive:

- internal hostnames
- real IP addresses
- VPN, VLAN, subnet, firewall, or routing details
- cloud account IDs, project IDs, tenant IDs, subscription IDs, or resource IDs
- diagrams generated from real environments

Use `private/`, `local/`, or another private repository for real configuration data.

## Reporting Security Issues

If you find a security issue in the harness itself, report it privately through the repository owner before opening a public issue.
